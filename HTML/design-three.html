Prototype Three

Interactive data visualisation

My data visualisation should be a combination of the two previous graphs. It should be able to change between the two data sets based on which button is pressed from the toolbox. Although the interaction is not extremely advanced or engaging, it still allows the user to filter through the data sets easily and without needing to scroll. The numbers needed for the data on the graph are on the axes, however, I would have optimally wanted it to be on top of each bar because this makes the graph easier to read and comprehend. The colour of the bars is gold. This is make the graph stand out and draw the user's eyes towards it.
